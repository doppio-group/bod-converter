BOD Converter







Instructions

  1. Select a .zap File
  2. Enter the BOD Noun. BOD Noun must start with a letter and cannot contain spaces
  3. Select the required BOD Verb
  4. Select the required BOD Direction
  5. Enter the BOD Prefix. This is optional. If empty, "M3BOD" will be used as the prefix
  6. Provide the Custom Envelope XML if needed. Click here for an example
  7. Click the Convert button and wait for a few seconds
  8. The output file will be downloaded
